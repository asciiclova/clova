## Install
AWS EC2の場合
### リポジトリの取得
$ sudo curl -sL https://rpm.nodesource.com/setup_10.x | bash -

### nodejsのインストール
$ sudo yum install -y nodejs

### nodeバージョン確認
$ node -v

### npmバージョン確認
$ npm --version

### ディレクトリ作成
mkdir time

### ディレクトリ移動
cd time

### 本プログラムのclone
git clone git@github.com:asciiclova/time.git

### 実行
node app.js
